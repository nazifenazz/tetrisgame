# Tetris Game

This project is a simple Tetris game made with Java.

I made this project to practice Java and basic game logic.

The game works in the terminal. There is also a small window for keyboard control.

The player can move the shape left and right. The player can also rotate the shape.

The shapes fall down automatically. When a row is full it can be cleared. If the new shape cannot fit the game ends.

## Features

- Random shape creation
- Moving shapes left and right
- Rotating shapes
- Automatic falling system
- Collision control
- Row clearing logic
- Game over control
- Terminal board view
- Simple keyboard control
- Different shape classes

## Technologies Used

- Java
- Swing
- IntelliJ IDEA

## How To Run

The project files are inside the zip file.

First the zip file should be opened.

Then the project can be opened with IntelliJ IDEA.

After opening the project the main game class can be run.

The game is controlled with keyboard keys.

## Files Shortly

- Tetris java file has the main game logic
- TetrisAnaSinifi java file starts the game
- HareketSekilleri java file is used for shape movement logic
- Kare java file is the square shape
- Cubuk java file is the line shape
- LHarfi java file is the L shape
- ZHarfi java file is the Z shape
- Ucgen java file is the triangle shape
- SatirSil java file clears full rows
- Main java file is a starter file from the project
